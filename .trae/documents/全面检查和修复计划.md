# 全面检查和修复计划

## 1. 问题概述

用户准备上线平台，要求确保页面指向真实数据而非模拟数据。当前系统存在以下核心问题：

### P0 必须修复
1. **互助任务发布使用硬编码用户信息** - 发布时 creatorName='我'，creatorAvatar 固定
2. **任务统计 API 返回 mock 数据** - profile/index.vue 的 loadTaskStats() 调用 tasksApi.getMyTasks() 返回 mockTasks
3. **活动 API 返回 mock 数据** - activitiesApi.getActivities() 返回 mockActivities

### P1 重要
4. **互助广场新用户显示模拟数据** - 应该显示空状态
5. **任务广场未登录也显示默认数据** - 应该显示空状态

## 2. 当前数据流分析

### 互助任务数据流
```
publish.vue → STORAGE_KEY ('ai_helper_tasks') → ai-helper/index.vue
publish.vue → MY_CREATED_TASKS_KEY ('ai_helper_my_created_tasks') → my-tasks.vue
```

### 个人中心任务统计
```
profile/index.vue → tasksApi.getMyTasks() → api.ts (返回 mockTasks)
```

### 3.2 修复任务统计返回真实数据

**文件**: `src/pages/profile/index.vue`

**问题**: loadTaskStats() 调用 API 返回 mockTasks

**修复**: 从 localStorage 直接读取并计算真实统计数据

```typescript
const MY_CREATED_TASKS_KEY = 'ai_helper_my_created_tasks'
const MY_ACCEPTED_TASKS_KEY = 'ai_helper_my_accepted_tasks'

const loadTaskStats = async () => {
  try {
    // 从 localStorage 读取真实任务数据
    const myCreatedTasks = JSON.parse(localStorage.getItem(MY_CREATED_TASKS_KEY) || '[]')
    const myAcceptedTasks = JSON.parse(localStorage.getItem(MY_ACCEPTED_TASKS_KEY) || '[]')

    // 计算统计数据
    const published = {
      all: myCreatedTasks.length,
      pending: myCreatedTasks.filter((t: any) => t.status === 'open').length,
      in_progress: myCreatedTasks.filter((t: any) => t.status === 'ongoing').length,
      completed: myCreatedTasks.filter((t: any) => t.status === 'completed' || t.status === 'pending_confirm').length
    }

    const accepted = {
      all: myAcceptedTasks.length,
      pending: myAcceptedTasks.filter((t: any) => t.status === 'open').length,
      in_progress: myAcceptedTasks.filter((t: any) => t.status === 'ongoing').length,
      completed: myAcceptedTasks.filter((t: any) => t.status === 'completed').length
    }

    taskStats.value = { published, accepted }
  } catch (error) {
    console.error('加载任务统计失败:', error)
  }
}
```

### 3.3 修复活动 API 返回真实数据

**文件**: `src/utils/api.ts`

**问题**: activitiesApi.getActivities() 返回 mockActivities

**修复**: 优先从 localStorage 读取，没有则返回空数组

```typescript
export const ACTIVITIES_STORAGE_KEY = 'linli_activities'

export const activitiesApi = {
  getActivities: (params?: { page?: number; limit?: number; status?: string; category?: string; sort?: string; order?: string }) => {
    if (MOCK_MODE) {
      return new Promise((resolve) => {
        setTimeout(() => {
          // 优先从 localStorage 读取真实数据
          try {
            const storedActivities = localStorage.getItem(ACTIVITIES_STORAGE_KEY)
            if (storedActivities) {
              const activities = JSON.parse(storedActivities)
              resolve({
                items: activities,
                pagination: {
                  page: params?.page || 1,
                  limit: params?.limit || 10,
                  total: activities.length,
                  totalPages: 1
                }
              })
              return
            }
          } catch (e) {
            console.error('读取活动数据失败', e)
          }
          // 如果 localStorage 没有，返回空数组（不是 mockActivities）
          resolve({
            items: [],
            pagination: {
              page: params?.page || 1,
              limit: params?.limit || 10,
              total: 0,
              totalPages: 1
            }
          })
        }, 300)
      })
    }
    return get<PaginatedData<Activity>>('/api/activities', params)
  },

  createActivity: (data: {...}) => {
    if (MOCK_MODE) {
      return new Promise((resolve) => {
        setTimeout(() => {
          const newActivity: Activity = {
            id: Date.now().toString(),
            user_id: '1',
            // ... 使用 userApi.getProfile() 获取当前用户信息
            user: { /* 从 localStorage 读取的用户信息 */ }
          }
          // 保存到 localStorage
          try {
            const storedActivities = localStorage.getItem(ACTIVITIES_STORAGE_KEY)
            const activities = storedActivities ? JSON.parse(storedActivities) : []
            activities.unshift(newActivity)
            localStorage.setItem(ACTIVITIES_STORAGE_KEY, JSON.stringify(activities))
          } catch (e) {
            console.error('保存活动数据失败', e)
          }
          resolve(newActivity)
        }, 300)
      })
    }
    return post<Activity>('/api/activities', data)
  }
}
```

### 3.4 修复互助广场新用户显示模拟数据

**文件**: `src/pages/ai-helper/index.vue`

**问题**: 新用户打开互助广场显示 defaultTasks 模拟数据

**修复**: 只在 localStorage 有数据时才显示，没有则显示空状态

```typescript
// 修改 loadFromStorage 函数
function loadFromStorage(key: string) {
  try {
    const stored = localStorage.getItem(key)
    if (stored) {
      return JSON.parse(stored)
    }
  } catch (e) {
    console.error(`加载数据失败: ${key}`, e)
  }
  return null  // 返回 null 而不是 defaultValue
}

// 使用
const tasks = ref<any[]>(loadFromStorage(STORAGE_KEY) || [])

// computed 修改
const filteredTasks = computed(() => {
  let result = [...tasks.value]

  // 如果没有数据，返回空数组
  if (result.length === 0) {
    return []
  }
  // ...
})
```
