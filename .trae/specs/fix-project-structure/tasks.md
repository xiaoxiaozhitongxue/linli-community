## [ ] 任务 2: 重构首页组件，替换为标准 HTML 元素

- **优先级**: P0
- **依赖**: 任务 1
- **描述**:
  - 将 `<view>` 替换为 `<div>`
  - 将 `<text>` 替换为 `<span>` 或 `<p>`
  - 将 `<scroll-view>` 替换为普通滚动容器
  - 将 `<image>` 替换为 `<img>`
  - 将 `<swiper>` 替换为简单的轮播实现
  - 修复点击事件处理
- **验收标准**: AC-1, AC-2
- **测试要求**:
  - 程序验证：页面可以正常渲染，无控制台错误
  - 人工判断：视觉展示和原来一致

## [ ] 任务 3: 重构状态管理和存储，移除 uni API

- **优先级**: P0
- **依赖**: 任务 1
- **描述**:
  - 将 `uni.getStorageSync` / `uni.setStorageSync` 替换为 `localStorage`
  - 将 `uni.navigateTo` / `uni.switchTab` 替换为 `router.push`
  - 将 `uni.showToast` 替换为自定义通知组件
  - 重构 store，使用纯 Vue 3 响应式
- **验收标准**: AC-1, AC-2, AC-3
- **测试要求**:
  - 程序验证：路由跳转正常，存储正常工作
  - 人工判断：用户操作有正确反馈
