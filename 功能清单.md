# 邻里社区APP 功能清单

## 🏠 首页 (Index)
**路由**: `/pages/index/index`
**功能**:
- 顶部搜索栏（跳转到搜索页面）
- 轮播图展示社区公告/热门活动
- 快捷入口导航（活动、互助、老人关怀、社区创业）
- 热门活动卡片列表
- 邻里动态feed流
- 悬浮发布按钮（展开菜单可发布动态/任务/活动）

## 🤝 互助广场 (AI Helper)

## 🔗 API 端点清单

### 基础端点
- `GET /api/hello` - 测试API
- `GET /api/health` - 健康检查
- `GET /api/test` - 测试接口

### 认证
- `POST /api/auth/login` - 用户登录

### 用户
- `GET /api/user/profile` - 获取用户信息
- `GET /api/user/posts` - 用户发布的帖子
- `GET /api/user/activities` - 用户的活动
- `GET /api/user/favorites` - 用户收藏
- `GET /api/users` - 用户列表

### 帖子
- `GET /api/posts` - 获取帖子列表
- `POST /api/posts` - 创建帖子
- `GET /api/posts/[id]` - 帖子详情
- `PUT /api/posts/[id]` - 更新帖子
- `DELETE /api/posts/[id]` - 删除帖子
- `POST /api/posts/[id]/like` - 点赞帖子
- `GET /api/posts/[id]/comments` - 获取评论
- `POST /api/posts/[id]/comments` - 发表评论
- `DELETE /api/posts/[id]/comments/[commentId]` - 删除评论

### 任务
- `GET /api/tasks` - 获取任务列表
- `POST /api/tasks` - 创建任务
- `GET /api/tasks/match` - AI匹配任务
- `GET /api/tasks/my` - 我的任务
- `GET /api/tasks/[id]` - 任务详情
- `POST /api/tasks/[id]/accept` - 接受任务
- `POST /api/tasks/[id]/complete` - 完成任务

### 活动
- `GET /api/activities` - 获取活动列表
- `POST /api/activities` - 创建活动
- `GET /api/activities/[id]` - 活动详情
- `POST /api/activities/[id]/join` - 参加活动
- `POST /api/activities/[id]/leave` - 取消参加
